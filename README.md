# Pavement_Management
Repo for Pavement Management System Codes for FHWA EAR Project "Cost-Efficient Network-level Pavement Management Framework for Maintenance"
This repository contains the code used in the study:
**"Impact of Integrating Use Stage into Network Level Life Cycle Planning"**  
## Contents
- `minnesota_case_study.ipynb` — Minnesota case study: agency and user cost 
  estimation based on MnDOT CHIP schedule over a 10-year period (2025–2034)
- `arizona_case_study.ipynb` — Arizona case study: network segmentation, 
  treatment selection under three prioritization strategies (Worst-First, 
  Preservation, Traffic-Weighted Preservation), and agency and user cost 
  estimation over a 10-year period using 2022 ADOT HPMS data
## Data Availability
**Arizona:** 2022 ADOT HPMS data is publicly available. The dataset used

**Minnesota:** Data was provided by MnDOT for research purposes only and
